# Quick Start

## Outcome

give AI coding hosts a sandbox-first codebase memory setup with install checks, indexing boundaries, evidence receipts, and recovery paths

## Two-Minute Path

1. Read `AGENTS.md`.
2. Copy the relevant host instruction into your AI coding host.
3. Run `06_EVALS/smoke_check.md`.
4. If the check fails, open `03_PITFALL_LOG.md`.
